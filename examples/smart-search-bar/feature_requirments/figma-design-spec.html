04 / Responsive — Mobile (≤ 768px)
Mobile Breakpoints
BreakpointLayout Change
≤ 768pxSwitch to 2-column product grid; full-width search bar
≤ 480pxStack chips horizontally scrollable; suggestions fullscreen overlay
Voice buttonOnly visible on mobile (≤ 768px); hidden on desktop
Mobile Interaction Notes
  • Search suggestions open as a full-screen overlay on mobile
  • Overlay has a dedicated back button (←) to dismiss
  • Recent searches shown by default when focused (before typing)
  • Chips row is horizontally scrollable with fade edge indicator
  • Product cards: 2 columns, 12px gap
  • Tap outside overlay to dismiss (with 200ms animation)
